🔗 API Endpoints

Single Message Send API

https://yourdomain.onrender.com/api/send?session=default&number=919876543210&text=Hello%20this%20is%20test

You can change session name (default, user2, user3, etc.)

Examples:

  • With Media:
    /api/send?session=user1&number=919876543210&text=Hello&media=https://image-url.jpg
  • Different Account:
    /api/send?session=rahul2&number=919555555555&text=Offer chal rahi hai
